# Restaurant Demo - Fine Dining Experience

A high-end, interactive restaurant website designed for modern luxury dining. Featuring a smooth interface, interactive menu, and seamless reservation system.

## 🌟 Features
- **Modern Minimalist Design**: A refined aesthetic for high-end establishments.
- **Interactive Menu**: Browse dishes with rich imagery and add to cart functionality.
- **Reservation System**: Real-time table selection and booking.
- **Mobile Optimized**: Fully responsive experience on all devices.
- **QR Scanner Integration**: Ready for in-restaurant digital ordering.

## 🚀 Getting Started

### Prerequisites
- Node.js installed on your machine.

### Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   npm install
   ```

### Development
Run the development server:
   ```bash
   npm run dev
   ```

### Build
Generate production-ready files:
   ```bash
   npm run build
   ```
